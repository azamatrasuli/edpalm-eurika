Ключевые решения

LLM Hybrid pre-fetch архитектура

RAG-поиск и профиль клиента загружаются до вызова LLM на бэкенде (детерминированно). LLM получает готовый контекст и работает только с 5 write-only инструментами через function calling.

1. Бэкенд: RAG search → top-5 чанков
2. Бэкенд: get_client_profile → профиль
3. Промпт: контекст + 5 tool definitions
4. LLM: генерация ответа / вызов tool

Tools 5 write-only инструментов

Только действия, которые нельзя выполнить детерминированно. Всё остальное — pre-fetch на бэкенде.

  • create_lead — создать лид в amoCRM
  • generate_payment_link — ссылка на оплату
  • create_manager_task — задача менеджеру
  • register_decline — зафиксировать отказ
  • escalate_to_manager — передать человеку

Speech STT/TTS через SpeechKit

Полная замена OpenAI Whisper и TTS. Фронтенд записывает WebM, бэкенд конвертирует в OGG через ffmpeg перед отправкой в SpeechKit.

WebM (фронт) → ffmpeg → OGG → SpeechKit STT
SpeechKit TTS → OGG → фронт (audio player)

RAG Переиндексация базы знаний

Текст KB не теряется — хранится в MD-файлах и в колонке content. Меняются только числовые векторы (1536d → 256d). pgvector + HNSW-индекс пересоздаётся.

  • knowledge_chunks: 746 чанков (sales + support)
  • memory_atoms: 582 факта (пересчёт из БД)
  • summaries: 203 саммари (пересчёт из БД)

Compliance-слой

Новые компоненты для соответствия ФЗ-152.

Согласие Экран до онбординга

  • 3 чекбокса: ПДн (обяз.), ИИ (обяз.), уведомления (опц.)
  • Несовершеннолетние: проверка возраста, форма для родителя
  • Логирование в consent_log
  • Отзыв через команду /stop или страницу профиля

Удаление Двухфазная схема

  • Soft delete сразу → данные скрыты из агента
  • Hard delete через 30 дней (cron-задача, 12 таблиц)
  • amoCRM: обезличивание контакта через API
  • Audit log: запись об удалении для РКН

Логи Маскирование ПДн

  • MaskingFilter: regex для телефонов (только production)
  • Whitelist безопасных ключей в tool arguments
  • 4 точечных правки: dms.py, renewal.py, telegram.py, chat.py

Дисклеймер ИИ-ассистент

  • Footer 11px над полем ввода (постоянный)
  • Первое сообщение при первом контакте
  • Описание Telegram-бота в BotFather
